1. Data Collection & Source

The Cut Nerd application follows the principle of data minimization. Cut Nerd only collects essential information authorized via official API protocols (OAuth 2.0). When you use Cut Nerd, the following data may be processed:

  • Basic profile identifiers (e.g., public display name, avatar) to identify your account within Cut Nerd.
  • Technical scopes required by Cut Nerd for video management and publishing.
  • Temporary access tokens used by Cut Nerd for automated workflow execution.

2. How Cut Nerd Uses Your Data

The data collected by Cut Nerd is processed exclusively for workflow automation and media synchronization. Cut Nerd does not engage in user behavior profiling, third-party tracking, or data monetization. All processing is strictly limited to the functional requirements of the Cut Nerd integration.

3. No Third-Party Disclosure

The Cut Nerd application maintains a strict Zero-Sharing Policy. Your data is never sold, leased, or distributed by Cut Nerd to third-party marketing entities. Access is limited only to the technical environment required for Cut Nerd to communicate with the destination platform's API.

4. Security and Encryption

All API keys and OAuth tokens used by Cut Nerd are stored using industry-standard encryption within a private server environment. Cut Nerd implements rigorous security measures to prevent unauthorized access and ensure the appropriate use of information.

5. Data Retention & Deletion

Users of Cut Nerd maintain full control over their data. Access to Cut Nerd can be revoked at any time through the security settings of the connected social network. Upon revocation, Cut Nerd renders all associated tokens invalid immediately and purges them from execution logs within 24 hours.
